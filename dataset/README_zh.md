# 数据集文档

[English](README.md) | 中文

本目录包含 TableAgent 项目的测试数据集及相关样本文件。

## 1. `samples.json` 字段说明

`samples.json` 文件定义了用于测试的样本数据，每个样本包含一个具体的表格 QA 任务。主要字段如下：

*   **`file_path`**：表格数据所在文件夹的路径。
    *   该路径指向包含任务所需表格文件的目录。
*   **`task`**：用户的自然语言查询或任务描述。
*   **`design`**：解决任务的设计方案，包括分步执行逻辑。
    *   **`type`**：任务类型（例如 `unknown` 或 `Fork-Join` 等）。
    *   **`checkout_list`**：解决任务的具体步骤列表。
        *   **`idx`**：步骤序号（从 0 开始）。
        *   **`info_item`**：此步骤中需获取的具体指令或信息。
        *   **`related_tables`**：完成此步骤所需的相关表格文件名列表。
        *   **`score_points`**：此步骤预期的关键结果或得分点（用于验证 Agent 的答案是否正确）。

## 2. 表格数据组织 (`tables` 目录)

`dataset/tables` 目录存储所有原始表格文件，按类别或来源组织。

*   **`tables/`**
    *   **`chinese_table/`**：存储中文表格数据集。
        *   包含多个以**特定主题或年份**命名的子文件夹（例如 `2007年土地成交与工程监理资质`、`2008年世界大学学术排名` 等）。
        *   每个子文件夹包含一个或多个关联的数据文件，通常为 `.xlsx`、`.xls` 或 `.csv` 格式。

### 目录结构示例

```text
dataset/
├── samples.json          # 样本任务描述文件
└── tables/
    └── chinese_table/    # 中文表格集合
        ├── 2007年土地成交与工程监理资质/
        │   ├── 1706800061649.xlsx
        │   └── 工程监理.xlsx
        ├── 【店铺数据】旗舰店运营报表12月-/

        │   ├── 【店铺数据】旗舰店运营报表12月-_CS.csv
        │   └── ...
        └── ...
```
